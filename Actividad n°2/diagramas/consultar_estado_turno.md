# Escenario de caso de uso 3

# Consultar estado de turno

## Descripción General
El usuario puede consultar el estado de su turno en tiempo real

## Flujo de eventos
* El usuario ingresa con su cuenta al sistema.
* El sistema verifica si los datos son correctos.
* El usuario selecciona el apartado de Turnos - Estado de turno.
* El sistema pide el código de seguimiento del turno.
* El usuario ingresa el código de seguimiento previamente dado al solicitar el turno.
* El sistema verifica el código.
* El sitio muestra el estado del turno del usuario.

 # Precondiciones:
 El usuario debe tener un turno generado.

# Postcondiciones:
Se muestra la información del turno.
